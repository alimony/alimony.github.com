These are unofficial .mobi builds of the official Django documentation, suitable for reading on a Kindle.

Just transfer the file to your device by Wifi, USB, or e-mail. For detailed instructions, see the Kindle support pages.

Details

The .mobi files have been generated by first building an .epub file in each corresponding branch of the Django repository:

git fetch upstream stable/1.6.x:stable/1.6.x
cd docs
make epub

And then running the .epub through KindleGen:

kindlegen _build/epub/Django.epub

However, this last step will currently fail:

Error(parsing):E3013: More number of characters are hidden using display:none than allowed limit. Limit: 10000  in file: 10000 line: 10000

This is fixed by using a separate Sphinx theme when building the .epub, one that leaves some sections out of the build. Getting this change into the Django master branch is currently in progress. For now, it can be used by downloading this diff and applying it to the checked out branch:

git apply django-epub-theme.diff

The above patch will also add a copyright page to the beginnings of the built .epub and .mobi since this is a requirement of the Django license.

Note:

These builds are experimental and work-in-progress. I just wanted to read the Django docs on my Kindle and thought others might too. Let me know how it works out for you, or if something is broken, so the builds can be improved.
